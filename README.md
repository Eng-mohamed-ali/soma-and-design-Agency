## Overview

SOMA Design & Marketing is a professional frontend showcase project created to present a modern digital agency brand through a clean, responsive, and visually polished web experience.

The project highlights how an agency can present its services, portfolio, training content, and contact channels in a way that feels credible, modern, and user-friendly across desktop, tablet, and mobile devices.

This repository is published as part of my portfolio to demonstrate my frontend development skills, design sense, component organization, and deployment workflow.

---

## Project Highlights

- Modern and professional agency-style interface
- Fully responsive layout for mobile, tablet, and desktop
- Smooth animations and interactive UI sections
- Clear navigation and multi-page structure
- Service presentation, portfolio showcase, and academy pages
- Contact-focused user experience with WhatsApp integration
- Deployed live with GitHub Pages
